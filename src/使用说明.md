
# RMSteg（原始固定分辨率）- 使用说明

## 简介

这是原始RMSteg的官方实现，固定输入分辨率224×224，使用官方预训练模型。

---

## 文件结构

```
src/
```

---

## 快速开始

### 1. 单张图片测试
```bash
python single_test.py
```
使用默认图片生成隐写并解码，结果保存在 `result/` 文件夹。

### 2. 批量测试
```bash
python all_test.py
```
对 `test_img/` 文件夹中的所有图片进行批量测试，结果保存在 `all_test_result/`。

### 3. 评估模型
```bash
python evaluate.py
```
计算SSIM、PSNR、LPIPS等质量指标，生成评估报告。

### 4. 训练模型
```bash
python train_single_gpu.py
```
单卡训练模型，保存权重到 `checkpoints/`。

### 5. 检测肉眼可见变化
```bash
python detect_visible_change.py
```
检测隐写后是否有肉眼可见的明显变化，结果保存为CSV。

## 各脚本功能详解

### `single_test.py`
- 功能：单张图片测试
- 输入：`test_img/1.png`
- 输出：
  - `result/final_host.png` - 原图
  - `result/final_steg.png` - 隐写后图片
  - `result/final_qr.png` - 原始QR
  - `result/final_trans_qr.png` - 变换后QR
  - `result/final_decode_qr.png` - 解码QR
  - `result/final_distort.png` - 失真后图片
  - `result/qr_error.png` - 误差图

### `all_test.py`
- 功能：批量测试 `test_img/` 文件夹的所有图片
- 输入：`test_img/0.png` ~ `50.png`
- 输出：`all_test_result/` 文件夹，每个图片生成完整结果
- 包含：`{n}_host.png`, `{n}_steg.png`, `{n}_qr.png`, `{n}_trans_qr.png`, `{n}_decode_qr.png`, `{n}_qr_error.png`, `{n}_distort.png`, `{n}_steg_res.png`

### `evaluate.py`
- 功能：评估模型性能
- 指标：
  - SSIM：结构相似性（越高越好）
  - PSNR：峰值信噪比（越高越好）
  - LPIPS：感知相似性（越低越好）
- 输出：`evaluation_results/evaluation_report.txt`

### `train_single_gpu.py`
- 功能：单卡训练
- 特点：
  - 自动寻找可用图片
  - 无需分布式设置
  - TensorBoard日志保存
  - 仅保存最终权重
- 输出：
  - `checkpoints/rmsteg_single_gpu_final.pth`
  - `log/rmsteg_single_gpu/`（TensorBoard日志）

### `detect_visible_change.py`
- 功能：检测隐写后是否有肉眼可见的明显变化
- 规则：预定义14张图片（1, 2, 3, 5, 15, 17, 25, 28, 29, 36, 39, 40, 48, 49）为有明显变化
- 输出：`detection_results.csv`

## 核心原理

### 1. IQRT（Invertible QR Transform）
对QR码进行可逆变换，使其风格与宿主图片匹配，提升视觉隐秘性。

### 2. ITF（Invertible Token Fusion）
自适应融合变换后的QR码和宿主图片的特征。

### 3. AttnFlow
使用注意力机制和归一化流，完成信息的嵌入与提取。

### 4. 失真模拟
模拟真实世界的失真（如JPEG压缩、打印等），提升模型鲁棒性。

## 注意事项

1. **预训练模型位置**：确保 `pretrained/rmsteg.pth` 存在
2. **图片命名**：测试图片命名为数字（0.png ~ 50.png）
3. **文件夹权限**：确保有写入权限保存结果
